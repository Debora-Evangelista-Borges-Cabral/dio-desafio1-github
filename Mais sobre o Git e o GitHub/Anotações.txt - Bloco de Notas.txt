Descomplicando:
Git e GitHub, são tecnologias diferentes que se complementam.

Git, software de versionamento de código distribuído.

Link para download do Git: https://git-scm.com/downloads

Curiosidade: O Git Bash é um terminal extendido para otimizar o uso do Git.

GitHub, plataforma em núvem de repositório remoto. 
Onde os desenvolvedores acessando a plataforma colaboram, contribuindo no código.
